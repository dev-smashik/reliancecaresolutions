# Reliance Care Solutions

A professional, compassionate home care service website built with **React**, **Vite**, and **Tailwind CSS**. This project features a modern, responsive design tailored for caregiving services in Northwest Ohio.

## 🚀 Quick Start

### Prerequisites
- [Node.js](https://nodejs.org/) (Recommended version v20+ or latest LTS)
- npm (Comes with Node.js)

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the local development server:
```bash
npm run dev
```
The application will be available at `http://localhost:5173`.

### Production Build
Generate a production-ready bundle:
```bash
npm run build
```
The output will be in the `dist/` directory.

## 🛠️ Technology Stack
- **Frontend Framework**: React 18
- **Build Tool**: Vite 6
- **Styling**: Tailwind CSS 4
- **Icons**: Lucide React & Custom SVG kit
- **Deployment**: Optimized for Vercel

## 📁 Project Structure
- `src/app/App.tsx`: Main application entry and root component.
- `src/assets/`: All images and brand assets (renamed to readable formats).
- `src/imports/`: Subsidiary components and standard design bundles.
- `src/styles/`: Global CSS and Tailwind configurations.
- `vercel.json`: Deployment configuration for Vercel (SPA routing).

## 🎨 Design & Assets

## 🌐 Deployment
This project is pre-configured for **Vercel**. 
- **Clean URLs**: Enabled.
- **SPA Routing**: Automatic redirects for client-side navigation.
- **Vite optimization**: Automatic asset hashing and compression.

Simply connect your repository to Vercel and it will handle the rest!
